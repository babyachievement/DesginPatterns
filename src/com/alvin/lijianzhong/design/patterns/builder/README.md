##意图（Intent）

>将一个复杂对象的构建与其表示相分离，使得同样的构建过程可以创建不同的表示。   ——《设计模式》GoF

##Builder模式中的几个要点

* Builder 模式主要用于“分步骤构建一个复杂的对象”。在这其中“分步骤”是一个稳定的算法，而复杂对象的各个部分（不是部分增加或减少，而是部分的细节发生变化）则经常变化。
